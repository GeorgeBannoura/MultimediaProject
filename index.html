What a Computer Engineering Student can work in future?


Computer Engineers works in hardware designing, developing and also hardware verification fields which include:

  • Microprocessors and Embedded Systems.
  • Data storage and memory.
  • IO devices.
  • Networking and data communication.
  • VLSI designing and coding.

Computer Engineers works in software designing and development field include:

  • Programming in high and low level languages.
  • Web applications and desktop application development.
  • Understanding Operating Systems.
  • General understanding of multimedia and computer graphics.

CE is a combination between Electrical Engineering and Computer Science

Here are some important study fields in CE the student will understand:

  • The basic ciruit design and alanysis.
  • How to work with binary and hex numbers better understanding of how computer takes instructions and execute programs.
  • The design and implementation using logic gates and binary system.
  • The basics of Signal Processing using digital and analog signals and their employment.
  • Essential Calculus and Physics skills.
  • Importand Electronics and control systems basics.
  • Object Oriented Programming (OOP) such as Java
  • Microcontrollers and Arduino Programming that makes hardware meets software.
  • Embedded systems design and implementation.
  • Programming using hardware descriptive languages such as Verilog.
  • Studying the Architecture of basic and advanced computer.
